## 百度站点收录

参考 [向百度主动推送网站链接](https://ifttl.com/push-urls-to-baidu/) 使用脚本定时推送网站链接到百度站点。对 push_to_baidu.sh 脚本的  parse 方法做了如下修改，以解决 xmllint 解析带有命名空间的 xml 文件报错的问题。 

```bash
function parse {
    local file=$1
    echo $file
    $XMLLINT --format --xpath "//*[local-name()='loc' and namespace-uri()='http://www.sitemaps.org/schemas/sitemap/0.9']/text()" "$file" | sed -e 's/https/\nhttps/g' > "$URL_TEMP"
    echo $URL_TEMP
}
```
